# Description of different folders
1. **9s_or_30s_which_to_take:** Shows analysis whether the features from 9s segments or the features from 30s segments are better.
2. **feature_extraction:** All the code used for feature extraction.
3. **fixed_feature_aggregation:** Brings together all the features together and creates train and test dataset.
4. **general_feature_extraction_code_with_instruction:** Detailed instructions to compute any feature for the whole dataset with any custom function that works only on a single audio segment.
5. **labels_calculation:** Analyzes the annotation (.rml) files and creates label for each segment.
6. **male_edf_segmentation:** Segments .edf files according to the given segment and overlap duration.
